## Сборка и установка

1. Убедитесь, что у вас установлен Python 3.6+.
2. Создайте и активируйте виртуальное окружение (рекомендуется):
   ```bash
   python -m venv venv
   source venv/bin/activate     # Linux/macOS
   venv\Scripts\activate        # Windows

3. В установке зависимостей убрана строка
    torch==1.10.2+cpu
    Версии вида +cpu (например 1.10.2+cpu) не публикуются на PyPI, а доступны только через специальные ссылки, предоставляемые на официальном сайте PyTorch.
    используйте команду ниже
    pip install torch==1.10.2+cpu -f https://download.pytorch.org/whl/torch_stable.html

5. Устанвовка зависимостей
    pip install -r requirements.txt
6.Запуск приложения
    Выполните из корня проекта:
    python main.py


## Использование
- Кнопка "Загрузить изображение" — загрузить изображение из файла.

- Кнопка "Включить камеру" и "Сделать снимок" — включить вебкамеру и сделать снимок, который остановит камеру.

- Опция выбора канала (R, G, B) — показывать только выбранный цветовой канал.

- Кнопка "Оттенки серого" — преобразовать изображение в оттенки серого.

- Ввод угла и кнопка "Повернуть" — повернуть изображение.

- Ввод координат и кнопка "Нарисовать прямоугольник" — нарисовать прямоугольник на изображении.

- Кнопка "Сбросить" — сбросить приложение к исходному состоянию.
